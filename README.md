Numerical guessing game

Простая числовая угадайка.
Позволяет ввести крайнее значение(диапозон) чисел, которые далее будут загаданы в случайном порядке.
Далее пользователю необходимо угадать загаданное число.
В результате правильного ввода загаданного числа, появится инфо о количестве попыток отгадывания
